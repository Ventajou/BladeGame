BladeGame

BladeGame is a HTML5 game library written for the Blade C# to JavaScript compiler.

It is also the subject of a series of articles I'm writing for BuildNewGames.com.

This site will present the articles and related demos in an easy to run manner, so you don't have to install anything to see what it's about.
